# flake8 report

Este archivo está reservado para el reporte de flake8.

Para generar el reporte de flake8, ejecuta el siguiente comando en la raíz del proyecto:

    flake8 . --format=html --htmldir=flake8-report

O para un reporte en texto plano:

    flake8 . > flake8-report.txt

El contenido de este archivo será reemplazado por los resultados del análisis de flake8.
